# 📱 Mina Appar

Här finns app‑liknande verktyg att använda via Safari.  
Lägg till dem på hemskärmen på din iPad för snabb åtkomst.

## ✅ Användning (iPad rekommenderas)

1. Öppna `index.html` i **Safari**
2. Tryck på **Dela‑ikonen**
3. Välj **”Lägg till på hemskärmen”**
4. Sidan fungerar nu som en app

Varje verktyg öppnas från startsidan och kan även sparas separat som egen genväg.
